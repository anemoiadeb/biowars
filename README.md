# ☣️ БИО-ВОЙНЫ — Setup Guide

## Файлы в проекте
```
bio_wars_bot.py   — весь бот в одном файле
requirements.txt  — зависимости
render.yaml       — деплой на Render.com
```

---

## 1. Получить токен бота
1. Открой @BotFather в Telegram
2. /newbot → введи имя и юзернейм
3. Скопируй токен вида `7xxxxxxxxx:AAF...`

---

## 2. Запуск на Pydroid 3 (локальный тест)

```bash
# Установить зависимости
pip install aiogram aiohttp

# Вставить токен в файл (строка TOKEN=) ИЛИ задать переменную
export BOT_TOKEN="7xxx:AAF..."

# Запустить
python bio_wars_bot.py
```

> WEBHOOK_URL не задан → бот автоматически работает в режиме polling.

---

## 3. Деплой на Render.com (продакшн)

### Шаг 1 — Репозиторий
Залей три файла в GitHub/GitLab репозиторий.

### Шаг 2 — Создать Web Service
1. https://render.com → New → Web Service
2. Подключи репозиторий
3. Render определит `render.yaml` автоматически

### Шаг 3 — Environment Variables
В настройках сервиса (Environment tab) добавь:

| Key          | Value                                         |
|--------------|-----------------------------------------------|
| `BOT_TOKEN`  | твой токен от BotFather                       |
| `WEBHOOK_URL`| `https://bio-wars-bot.onrender.com` (твой URL)|

### Шаг 4 — Deploy
Нажми Deploy. Render установит зависимости и запустит бот.

> ⚠️ Free tier засыпает после 15 мин простоя.
> Чтобы этого избежать — подключи UptimeRobot (бесплатно) для пинга каждые 5 мин:
> https://uptimerobot.com → New Monitor → HTTP → URL твоего Render сервиса.

## 5. Игровые механики

### Заражение
- Базовый шанс: **40%**
- +5% за каждый уровень Заразности атакующего
- −7% за каждый уровень Резистентности цели
- −15% за каждую Вакцину у цели
- Кулдаун: **30 минут**

### Лаборатория
- Работа даёт `10 + lab_level × 5` патогенов (+случайный бонус)
- Кулдаун: **1 час**
- Пассивный доход: `(2 + lab_level × 0.5)🧪 / час`
- Апгрейд лабы: `lab_level × 80🧪`, макс ур. **10**

### Магазин
| Улучшение      | Стоим. за ур.  | Макс  | Эффект                       |
|----------------|----------------|-------|------------------------------|
| Заразность     | 40🧪 × ур.     | 5     | +5% к шансу заражения        |
| Летальность    | 50🧪 × ур.     | 5     | +6% патогенов украдено       |
| Резистентность | 60🧪 × ур.     | 5     | −7% к входящему заражению    |
| Вакцина        | 30🧪 / штука   | 3     | −15% к входящему заражению   |

### Уровни БК
- Каждый уровень требует `level × 100` XP
- XP получают за: работу в лабе (+15), успешное заражение (+20), лечение (+10)
